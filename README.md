# smartsymptom-ai
SmartSymptom AI is an AI-powered healthcare assistant that analyzes user-entered symptoms and provides possible medical explanations and guidance. Built with Python, FastAPI, and AI models, it offers a simple web interface to help users understand symptoms and decide when to seek medical care.
## Features
- AI symptom analysis
- FastAPI backend
- Web interface
- Real-time AI responses

## Technologies Used
- Python
- FastAPI
- OpenAI API
- HTML / JavaScript

## How It Works
1. User enters symptoms
2. AI analyzes symptoms
3. The system returns possible medical explanations and guidance

## Disclaimer
This tool provides informational guidance only and does not replace professional medical advice.
